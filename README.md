# ✈️ Horizons - Voyage Planner

Voyage Planner is a modern travel itinerary planning application built with React, TypeScript, and TailwindCSS.
It allows users to create trips, organize daily activities in a timeline, and track travel budgets in a clean and intuitive interface.

The project focuses on thoughtful UX, modern frontend architecture, and scalable component design.

---

## 🚀 Live Demo

[Voyage Planner](https://travel-itinerary-app-alpha.vercel.app)

---

## ✨ Features

### 🧳 Trip Management

- Create, edit, and delete itineraries
- Track destination, dates, travelers, and trip notes
- Cover images for each trip

### 📅 Timeline-Based Planning

- Activities organized by trip day
- Add, edit, and remove activities
- Optional time scheduling
- Support for locations, reservations, and notes

### 💰 Budget Tracking

- Track planned spending per activity
- Trip-level budget overview
- Remaining budget calculation
- Visual budget progress indicator

### 🎨 Modern UI

- Built with Radix UI + TailwindCSS
- Accessible components
- Sticky navigation and timeline layout
- Responsive design

### 🔔 UX Enhancements

- Confirmation dialogs for destructive actions
- Toast notifications for user feedback
- Clean layout with shared header/footer
